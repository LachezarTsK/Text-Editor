To run the application, click here: https://lachezartsk.github.io/Text-Editor/

The Text Editor has the following functionalities:<br/>
- count number of characters<br/>
- count number of words *(see comment below)<br/>
- to upper case<br/>
- to lower case<br/>
- clear all<br/>
- copy to clipboard<br/>
- clear space<br/>
- reverse characters within each character group **(see comment below)<br/>
- reverse whole text<br/>
- undo<br/>
- redo

//* Word is defined as any group of printable characters that contain at least one letter or one digit. 
The method can be easily adjusted according to whatever definition of word is chosen.

Different text editors on the internet have different word definitions: 
some define it as any group of printable characters, others, 
such as https://wordcounter.net (as of April 2026), define word as any group of printable characters 
that has at least one letter or digit. The definition of the latter is chosen for this application.

** Character group is each non-sepaparted printable charatcers. The character group could be either a word, as defined above (such as w@, 12&, w1, etc)
or just a bunch of other characters (such as !@#$, !!??, etc)
